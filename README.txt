A custom registration form plugin built for the Living Heritage Nursery School to handle new student applications.

== Description ==

This plugin provides a complete registration form solution. It includes:
* A frontend form displayed via a shortcode `[living_heritage_registration_form]`.
* A custom database table to securely store all submissions.
* An admin dashboard menu to view and manage submissions.
* Email notifications for new submissions.

== Installation ==

1. Upload the `living-heritage-forms` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place the shortcode `[living_heritage_registration_form]` on the page where you want the form to appear.
